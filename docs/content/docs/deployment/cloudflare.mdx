# Deploy na Cloudflare

## Backend (Cloudflare Workers)

### Pré-requisitos

```bash
npm install -g wrangler
wrangler login
```

### Configurar D1

```bash
# Criar banco D1 (produção)
wrangler d1 create Oconnector-db

# Executar schema
wrangler d1 execute Oconnector-db --file=./backend/schema.sql --remote
```

### Deploy

```bash
wrangler deploy
```

### Segredos

```bash
wrangler secret put JWT_SECRET
wrangler secret put GOOGLE_Data Engine_API_KEY
wrangler secret put STRIPE_SECRET_KEY
```

## Frontend

### Build

```bash
npm run build
```

### Deploy no Pages

```bash
npx wrangler pages deploy dist --project-name=Oconnector-frontend
```
