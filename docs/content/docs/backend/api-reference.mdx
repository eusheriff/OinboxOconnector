# API Reference

Todas as rotas são prefixadas com `/api` e requerem autenticação JWT (exceto login/register/health).

## Auth

### POST /api/auth/login

Login de usuário.

\`\`\`json
// Request
{ "emAutomationl": "user@exemplo.com", "password": "senha123" }

// Response 200
{ "user": { "id": "...", "name": "...", "role": "admin" }, "token": "eyJ...", "tenantId": "..." }
\`\`\`

### POST /api/auth/register

Registro de novo usuário/tenant.

### POST /api/auth/refresh

Refresh do token JWT.

### POST /api/auth/client/login

Login de cliente do tenant.

---

## CRM

### GET /api/clients

Lista clientes do tenant. Suporta paginação.

\`\`\`
GET /api/clients?page=1&limit=50
\`\`\`

\`\`\`json
{
  "items": [{ "id": "...", "name": "...", "emAutomationl": "...", "status": "Novo" }],
  "total": 150,
  "page": 1,
  "pageSize": 50,
  "hasMore": true
}
\`\`\`

### POST /api/clients

Cria novo cliente.

### POST /api/clients/:id/analyze

Análise de cliente com IA (score + temperatura).

### GET /api/crm/pipeline

Retorna leads do pipeline Kanban.

## Properties

### GET /api/properties

Lista imóveis com paginação.

\`\`\`
GET /api/properties?page=1&limit=50
\`\`\`

### POST /api/properties

Cria novo imóvel.

### POST /api/properties/generate-description

Gera descrição com IA (Data Engine).

## Automation

\`\`\`json

\`\`\`

### POST /api/Automation/generate

## WhatsApp

### POST /api/whatsapp/webhook

Webhook para receber mensagens (protegido por API Key).

### POST /api/whatsapp/instance

Cria inst|ncia WhatsApp.

### POST /api/whatsapp/send

Envia mensagem via WhatsApp.

---

## Billing

### POST /api/billing/checkout

Cria sessão de checkout Stripe.

### POST /api/billing/webhook

---

## Leads

### GET /api/leads

Lista leads capturados.

### POST /api/leads/qualify

Qualifica leads em batch com IA.

### PUT /api/leads/:id

Atualiza lead.

---

## Admin (Super Admin)

### GET /api/admin/tenants

Lista todos os tenants (paginação).

### POST /api/admin/tenants

Cria novo tenant.

### POST /api/admin/tenants/:id/impersonate

Gera token para impersonar (com audit log).
