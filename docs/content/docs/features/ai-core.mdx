# Automation Core

Oinbox utiliza Google Data Engine 2.5 Flash para recursos de IA:

- **Voice-to-CRM**: Transcrição de áudio para notas
- **Visão Computacional**: Análise de fotos de imóveis
- **Chatbot Especialista**: Respostas inteligentes para leads
