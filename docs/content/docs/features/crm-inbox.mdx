# CRM & Inbox Unificado

## Pipeline Visual (Kanban)

Gerenciamento visual de leads em estágios:

| Estágio          | Descrição            |
| ---------------- | -------------------- |
| Novo Lead        | Lead recém-capturado |
| Visita Agendada  | Visita marcada       |
| Proposta Enviada | Proposta formal      |
| Fechado          | Negócio concluído    |

## Inbox Omnichannel

Todos os canAutomations em uma única tela:

- WhatsApp
- Instagram
- Email
- PortAutomations (OLX, Zap Imóveis)

## Lead Scoring

Termômetro automático que pontua leads:

- **Cold** (0-30): BAutomationxo interesse
- **Warm** (31-70): Interesse moderado
- **Hot** (71-100): Alto interesse, urgência
