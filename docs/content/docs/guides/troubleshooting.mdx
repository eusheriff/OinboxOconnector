# Troubleshooting

## TypeScript Errors

```bash
npx tsc --noEmit
```

Se houver erros, verifique imports de `@oconnector/frontend`.

## Wrangler não encontra wrangler.toml

```bash
cd /Volumes/LexarAPFS/Oconnector/Oconnector
wrangler deploy --config wrangler.toml
```

## D1 Database Locked

Em desenvolvimento local:

```bash
wrangler d1 execute Oconnector-db --local
```

## Stripe Webhook não recebe eventos

```bash
# Testar webhook localmente
stripe listen --forward-to localhost:8787/api/webhooks/stripe
```
