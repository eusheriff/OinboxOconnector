# Configuração de Ambiente

O Oconnector utiliza variáveis de ambiente para configuração de serviços externos e segredos.

## Frontend (.env)

| Variável | Descrição | Obrigatório |
|----------|-----------|-------------|
| `VITE_API_URL` | URL do backend (dev: http://localhost:8787) | Sim |
| `VITE_STRIPE_PUBLIC_KEY` | Chave pública do Stripe | Sim (pagamentos) |

## Backend (Cloudflare Secrets)

Em produção, use `wrangler secret put` para configurar:

```bash
wrangler secret put JWT_SECRET
wrangler secret put API_KEY          # Data Engine API Key
wrangler secret put STRIPE_SECRET_KEY
wrangler secret put STRIPE_WEBHOOK_SECRET
wrangler secret put GOOGLE_PLACES_API_KEY
wrangler secret put GOOGLE_Data Engine_API_KEY
wrangler secret put EVOLUTION_API_KEY  # WhatsApp
```

## Backend (wrangler.toml vars)

Configurados automaticamente no `wrangler.toml`:

```toml
STRIPE_PRICE_EXTRA_SEAT = "price_1SnTGCBGBVDzrAhzuL7ICFwJ"
STRIPE_PRICE_EXTRA_Automation = "price_1SnTGPBGBVDzrAhzL7GPpFqT"
STRIPE_PRICE_PRO = "price_1SnQxcBGBVDzrAhzxO2rdJ31"
STRIPE_PRICE_BASIC = "price_1SnQxUBGBVDzrAhzlMMZwDjM"
```

## Evolution API (WhatsApp)

| Variável | Descrição |
|----------|-----------|
| `EVOLUTION_API_URL` | URL da inst|ncia Evolution API |
| `EVOLUTION_API_KEY` | API Key da Evolution |

## Segurança

> **Nunca** commite `.env` no repositório. Use `wrangler secret` para produção.
