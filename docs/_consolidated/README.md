# Documentação Consolidada - Oinbox

Este diretório centraliza todo o conhecimento técnico e operacional do projeto Oinbox.

## Estrutura

- **00_overview/**: Visão geral do sistema e legados.
- **01_architecture/**: Decisões arquiteturais (ADRs) e diagramas.
- **02_runbooks/**: Manuais operacionais (Deploy, Desenvolvimento Local).
- **03_api/**: Documentação de referência da API REST.
- **04_audit/**: Relatórios de auditoria e qualidade.
- **decisions/**: Registro de Decisões Arquiteturais (ADRs).
- **99_archive/**: Arquivos obsoletos ou históricos.

## Como contribuir

1. **Não crie docs na raiz**. Escolha a pasta mais apropriada aqui.
2. Mantenha os docs atualizados junto com o código.
