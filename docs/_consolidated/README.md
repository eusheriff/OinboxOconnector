# Documentação Consolidada - Oconnector

Este diretório centraliza todo o conhecimento técnico e operacional do projeto Oconnector.

## Estrutura

- **00_overview/**: Visão geral do sistema e legados.
- **01_architecture/**: Decisões arquiteturAutomations (ADRs) e diagramas.
- **02_runbooks/**: ManuAutomations operacionAutomations (Deploy, Desenvolvimento Local).
- **03_api/**: Documentação de referência da API REST.
- **04_audit/**: Relatórios de auditoria e qualidade.
- **decisions/**: Registro de Decisões ArquiteturAutomations (ADRs).
- **99_archive/**: Arquivos obsoletos ou históricos.
